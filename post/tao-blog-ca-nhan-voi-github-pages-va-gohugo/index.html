Tạo blog cá nhân với Github Pages và Hugo

Aug 30, 2019 - github hugo developer

Bạn muốn có một trang blog cá nhân để chia sẻ kiến thức, kinh nghiệm và mọi niềm vui buồn trong cuộc sống. Trên internet hiện có rất rất nhiều cách để tạo ra một trang blog đơn giản và miễn phí. Trong bài viết này mình sẽ hướng dẫn cách tạo một trang blog cá nhân bằng Github Pages và Hugo.

Github Pages và Hugo là gì ?

  • Github Pages là dịch vụ lưu trữ wesite trực tiếp thông qua Github.
  • Hugo là một trình tạo trang tĩnh nhanh và hiện đại được viết bằng Go và được thiết kế để tạo trang web nhanh tróng và vui vẻ.

Nói ngắn gọn thì ta sẽ sử dụng Hugo để tạo ra một trang web tĩnh, sau đó đẩy nó lên Github và cấu hình để cho chạy như là một trang web.

Lưu ý:

  • ở bài viết này mình sử dụng macOS để hướng dẫn nhé. Các hệ điều hành khác cũng tương tự. Xem thêm ở đây.
  • Để thực hiện được những bước dưới đây bạn cũng cần phải có một chút kiến thức về website, dòng lệnh và git nhé.

Cài đặt và tạo blog

Bước 1: Cài đặt Hugo

Homebrew, Một trình quản lý gói cho macOS, có thể được cài đặt từ brew.sh . Xem cài đặt nếu bạn đang chạy Windows, v.v.

brew install hugo

Xác minh cài đặt:

hugo version

Bước 2: Tạo một trang mới

hugo new site gocuatoi

Lệnh trên sẽ tạo một trang web Hugo mới với thư mục có tên là gocuatoi

Bước 3: Thêm một chủ đề (theme)

Xem Themes.gohugo.io và tìm ra một chủ đề bạn mà bạn muốn sử dụng nó cho blog của mình. Trong hước dẫn này sử dụng Manis Hugo Theme đơn giản mà đẹp (đối với mình). Chủ đề này hiện đang được mình sử dụng cho chính trang blog này (có chỉnh sửa lại chút ít).

cd gocuatoi
git init
git submodule add https://github.com/yursan9/manis-hugo-theme themes/manis
cp themes/manis/exampleSite/config.toml config.toml

Bước 4: Tạo bài viết đầu tiên

hugo new post/bai-viet-dau-tien.md

Nó sẽ tạo ra một tệp markdown là /content/post/bai-viet-dau-tien. Nội dung tệp sẽ trông như thế này:

---
title: "Bai Viet Dau Tien"
date: 2019-08-1T08:38:24+07:00
draft: true
---

Giờ thì bạn có thể chỉnh sửa nội dung tệp nếu muốn.

Bước 5: Khởi động Hugo Server

Chạy lệnh sau đây để khởi động Hugo Server trước khi viết bài:

hugo server -D

Đi đến trang web mới của bạn tại http://localhost:1313. Giò bạn có thể thoải mái chỉnh sửa nội dung của bạn và xem kết quả ngay trực tiếp trên trình duyệt. Khi bạn sửa thì trình duyệt sẽ tự động làm mới trang để cập nhật nội dung bài viết.

Bước 6: Cá nhân hóa

Cần phải chỉnh sửa một chút đề cá nhân hóa web. Mở tệp config.toml và cập nhật những trường sau:

baseURL = "https://example.com/"
languageCode = "vi-vn"
title = "Blog cá nhân của tôi"
theme = "manis"

Thay thế title ở trên với tiêu đề mà bạn muốn. Ngoài ra, nếu bạn đã có sẵn một tên miền, hãy đặt baseURL. Lưu ý rằng giá trị này là không cần thiết khi chạy Hugo Server khi viết bài dưới local.

Để biết các tùy chọn cấu hình cụ thể của chủ đề, xem trang web chủ đề.

Bước 7: Xây dựng trang tĩnh

Đơn giản chỉ cần chạy lệnh:

hugo

Trang web cùa bạn sẽ được đặt ở thư mục /public Giờ bạn có thể upload toàn bộ tệp trong thư mục /public lên bất cứ hosting nào để chạy trang web của bạn.

Ok. Đã chuẩn bị xong trang tĩnh, giờ sẽ qua bước đưa web của bạn lên internet để người khác có thể truy cập được.

Deploy lên Github Pages

Bước 1: Tạo Github Respository

Truy cập vào đây để tạo một respository mới với tên là gocuatoi.github.io. Lưu ý đây sẽ là tên miền tuy cập trang web của bạn.

Bước 2: Push source lên Github

cd gocuatoi/public
git init
git add .
git commit -m "Bai viet dau tien"
git remote add origin "REMOTE_RESPOSITORY_URL"
git push -u origin master

Lưu ý: nhớ thay REMOTE_RESPOSITORY_URL bằng địa chỉ repo của bạn.

Bước 3: Thưởng thức

Tới đây là đã hoàn tất rồi, bạn có thể truy cập vô web của bạn và thưởng thức. Ở đây thì địa chỉ web của mình là https://gocuatoi.github.io.

Linh tinh

Tên miền tùy chỉnh

Nếu bạn không thích tên miền name.github.io thì có thể sử dụng tên miền tùy chỉnh của bạn ví dụ như gocuatoi.com thì bạn có thể truy cập vào đây để thêm tên miền của bạn.

Triển khai nhanh

Bình thường là cứ mỗi lần bạn có thêm hay cập nhật bài viết thì thường bải lặp lại các bước như build rồi commit như vậy sẽ rất nhàm chán. Vậy nên bạn sẽ cần đoạn script này để có thể chỉ cẩn “Enter” một phát là xong luôn.

Tệp deploy.sh:

#!/bin/sh

# If a command fails then the deploy stops
set -e

printf "\033[0;32mDeploying updates to GitHub...\033[0m\n"

# Build the project.
hugo # if using a theme, replace with `hugo -t <YOURTHEME>`

# Go To Public folder
cd public

# Add changes to git.
git add .

# Commit changes.
msg="rebuilding site $(date)"
if [ -n "$*" ]; then
    msg="$*"
fi
git commit -m "$msg"

# Push source and build repos.
git push origin master

Lưu tệp ở thu mục gốc của thư mục web Hugo. Và chạy lệnh sau

chmod +x deploy.sh

Ok. Cứ mỗi lần thêm bài mới hay thay đổi gì chỉ cần chạy lệnh sau là nó sẽ tự động làm mọi thứ cho bạn.

./deploy.sh

Tổng kết

Cũng khá là đơn giản để tạo một trang blog đúng không. Hy vọng bài viết này sẽ hữu ích đối với mọi người.
