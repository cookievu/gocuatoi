Coc - Intellisense engine cho Vim/Neovim

Nov 03, 2019 - vim neovim coc Intellisense lsp

Mở đầu

Trước khi đi vào nội dung bài viết thì ta cần biết được Intellisense engine là gì nhé.

IntelliSense is a general term for a variety of code editing features including: code completion, parameter info, quick info, and member lists. IntelliSense features are sometimes called by other names such as “code completion”, “content assist”, and “code hinting”.

Nói cho dễ hiểu thì IntelliSense là một thuật ngữ nói về tính năng hỗ trợ cho dev thực hiện việc code dễ thở hơn như khả năng hoàn thành mã, hỗ trợ nội dung, gợi ý mã, …

Cho bạn nào chưa biết, lý do VSCode nổi tiếng và phổ biến đến thế là bởi vì nó có một bộ IntelliSense Engine cực cực xịn sò. Nó có khả năng hỗ trợ nhiều loại ngôn ngữ rất hiệu quả đặc biệt là javascript/typescript. Lý do nó xịn như vậy cũng đơn giản là vì nó được làm bởi những lập trình viên giỏi Microsoft. Nếu VSCode thiếu đi cái IntelliSense này thì có lẽ nó không được nổi tiếng và phổ biến như bây giờ.

Rất tiếc là mình không thích sài VSCode nên thật tiếc là mình không thể sử dụng được cái IntelliSense sịn sò đó. Nhưng mình đã tìm được một cái Intellisense xịn không kém đó là Coc.nvim.

Coc là một intellisense engine cho Vim/Neovim.

Coc là một plugin cực kì xịn luôn. Nó hỗ trợ rất là nhiều ngôn ngữ như javascript, typescript, golang, php, … và cài đặt các ngôn ngữ bổ sung rất dễ dàng mà không cần phải config nhiều chỉ bằng câu lệnh :CocInstall [name].

Cài đặt

Coc cần có nodejs để hoạt động nhé.

curl -sL install-node.now.sh/lts | bash

Cài đặt bằng vim-plug

" Use release branch (Recommend)
Plug 'neoclide/coc.nvim', {'branch': 'release'}

" Or latest tag
Plug 'neoclide/coc.nvim', {'tag': '*', 'branch': 'release'}
" Or build from source code by use yarn: https://yarnpkg.com
Plug 'neoclide/coc.nvim', {'do': 'yarn install --frozen-lockfile'}

Khởi động lại Vim/Neovim và chạy :PlugInstall.

Cài thêm extension.

Truy cập đường dẫn Using coc extensions để xem thêm các extension về ngôn ngữ mà bạn sử dụng.

Mình thì sử dụng những extension này: coc-tsserver, coc-html, coc-css, coc-angular. Để cài đặt extension thì chỉ cần chạy :CocInstall [name of extension] là được.
