Setup môi trường lập trình PHP trên macOS

Nov 07, 2019 - macos php

Để thiết lập một môi trường lập trình PHP hiện tại rất là dễ dàng. Chỉ cần bạn lên google và tìm kiếm một chút là có một đống những bài viết hướng dẫn rất ngắn gọn và nhanh chóng. Chỉ cần bỏ và chưa tới 1 phút mà đã có thể bắt đầu code được luôn rồi. Thế nên trong bài viết này mình sẽ không hướng dẫn setup môi trường php không phải theo những cách đơn giản và nhanh gọn đó, mình sẽ hướng dẫn theo cách “try hard” hơn =)).

Homebrew

Mọi người dùng macOS chắc không lạ gì Homebrew nữa đúng không.

Homebrew installs the stuff you need that Apple (or your Linux system) didn’t.

Cài đặt:

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Kiểm tra cài đặt:

~ brew --version
Homebrew 2.1.16
Homebrew/homebrew-core (git revision bbde; last commit 2019-11-06)
Homebrew/homebrew-cask (git revision accd5; last commit 2019-11-07)

Nginx

Mặc định thì macOS đã cài sẵn apache rồi nhưng mình lại không thích sài apache nên mình sẽ dùng nginx. Đa phần các hệ thống máy chủ trên thế giới đều sử dụng nginx làm websever bởi tính ổn định và hiệu xuất cao của nó.

brew install nginx

Sau khi cài đặt thì khởi động nginx:

brew services start nginx

Lệnh này sẽ giúp cho nginx sẽ auto run mỗi khi bạn mở máy.

Mở trình duyệt và truy cập địa chỉ http://localhost:8080 và nếu thấy trình duyệt hiển thị như hình dưới là ok.


PHP

Mặc định macOS cũng đã có PHP đi kèm nhưng mình luôn muốn có bản PHP mới nhất nên cũng sẽ cài mới luôn.

brew install php@7.3

Mặc dù bạn đã cài PHP mới rồi nhưng hệ thống vẫn còn sử dụng bản PHP cũ nên mình sẽ phải link bản PHP mới cài.

brew link --force php@7.3

Kiểm tra xem đã link thành công hay chưa.

MariaDB

MariaDB là một phiên bản cải tiến của Mysql vì thế mà MariaDB ngon hơn nhiều. Do vậy mình cũng chọn MariaDB làm cơ sở liệu chính luôn.

brew install mariadb

Khởi động MariaDB

brew services start mysql

Truy cập vào MariaDB console để đặt mật khẩu cho tài khoản root:

mariadb
> use mysql;
> ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'root password';
> FLUSH PRIVILEGES;
> quit;

Cuối cùng là đăng nhập vào MariaDB bằng password vừa mới tạo.

mysql -u root -p

Dnsmasq

Dnsmasq là một phần mềm giúp cho việc cấu hình DNS tự động. Giúp cho mình có thể giả lập bất cứ tên miền nào mà chúng ta muốn, tất nhiên là chỉ ở local thôi nhé. Ở đây mình sẽ dùng Dnsmasq để hỗ trợ cho phần virtual host.

Đầu tiên là cài đặt:

brew install dnsmasq

Thêm tên miền .test vào dnsmasq:

echo 'address=/.test/127.0.0.1' > /usr/local/etc/dnsmasq.conf

Add domain vào resolver

sudo mkdir -v /etc/resolver
sudo bash -c 'echo "nameserver 127.0.0.1" > /etc/resolver/test'

Khởi động Dnsmasq (lưu lý là lần này có sudo nhé):

sudo brew services start dnsmasq

Ping một tên miền bất kỳ để kiểm tra:
